# シンプル電卓
このアプリは、「JavaScript製電卓アプリ」です。基本的な四則演算（足し算・引き算・掛け算・割り算）に対応し、視覚的にもわかりやすい UI を実現しています。

![screenshot](images/calc.jpg) 


# 概要
- C（クリア）機能
- ←（1文字削除）機能
- 0除算や無効入力時の Error 表示
- 負の数や連続演算（例：6×-3）に対応

# 使用技術
- HTML5
- CSS3（Flexbox, 背景画像）
- JavaScript（ES6クラスベース）
